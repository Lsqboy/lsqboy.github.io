Android sdk7.0 正式发布了。新特性可不少。

## 重要的开发者功能

1. 多窗口支持
2. 通知增强功能
3. 个人资料指导的 JIT/AOT 编译
4. 快速的应用安装路径
5. 随时随地低电耗模式
6. 后台优化
7. Data Saver
8. Vulkan API
9. Quick Settings Tile API
10. 号码屏蔽
11. 来电过滤
12. 区域设置和语言
13. 新增的表情符号
14. Android 中的 ICU4J API
15. OpenGL ES 3.2 API
16. Android TV 录制
17. Android for Work
18. 无障碍功能
19. 直接启动
20. 密钥认证
21. 网络安全性配置
22. 默认受信任的 CA
23. APK Signature Scheme v2
24. 作用域目录访问
25. 键盘快捷键辅助工具
26. 持续性能 API
27. VR 支持
28. 打印服务增强
29. 虚拟文件
30. FrameMetricsListener API
